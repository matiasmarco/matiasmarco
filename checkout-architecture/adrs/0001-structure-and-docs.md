# ADR 0001: Estructura base de documentación

- **Estado:** Aceptado
- **Fecha:** 2026-04-11

## Contexto
Se requiere una base homogénea para documentación técnica y funcional.

## Decisión
Adoptar directorios dedicados para:
- Markdown
- Diagramas
- OpenAPI
- Ejemplos JSON
- ADRs

## Consecuencias
- Mejor discoverability.
- Onboarding más rápido.
- Menor dispersión documental.
